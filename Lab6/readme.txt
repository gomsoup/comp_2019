1. 개요
Recursive Descent Parser로 짜여진 간단한 사칙연산 계산기입니다.
+ -의 기능만을 수행할 수 있습니다. 또한 한자리 숫자만 지원합니다.
괄호와 연산 우선순위가 적용됩니다.

2. 컴파일 및 사용
본 소스코드는 다음과 같은 명령으로 컴파일 할 수 있습니다.

	gcc main.c -o main

이후 다음과 출력된 바이너리를 실행시켜 사칙연산을 입력하면 계산 결과가 출력됩니다.
동봉된 샘플 파일은 다음과 같이 사용할 수 있습니다
	
	./main < 1.inp

3. 문법
본 프로그램은 다음과 같은 문법을 따릅니다.

	E -> T E'
	E' -> e
	E' -> + T E'
	T' -> e
	T' -> * F T'
	F -> ( E )
	F -> number

이 때 e는 빈 문자열을, number는 입력된 숫자를 의미합니다.
